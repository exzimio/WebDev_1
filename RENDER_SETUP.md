# 🍕 Mega Pizza - Render.com Setup

## Instruções de Deploy no Render.com

### 1️⃣ **Vai a Render.com**
https://render.com

### 2️⃣ **Clica em "New +"** (canto superior direito)

### 3️⃣ **Escolhe "Web Service"**

### 4️⃣ **Conecta GitHub**
- Clica em **"Connect GitHub account"**
- Autoriza o Render a aceder aos teus repositórios

### 5️⃣ **Seleciona o repositório**
- Procura por `WebDev_1`
- Clica em **"Connect"**

### 6️⃣ **Configura o serviço:**

| Campo | Valor |
|-------|-------|
| **Name** | mega-pizza |
| **Environment** | Node |
| **Build Command** | `npm install` |
| **Start Command** | `npm start` |
| **Instance Type** | Free |

### 7️⃣ **Clica em "Create Web Service"** ✅

### 8️⃣ **Espera o deploy** (3-5 minutos)

### 9️⃣ **O site estará em:**
```
https://seu-nome.onrender.com
```

---

## 📝 Configurações extras (opcionais)

### Deploy automático
Cada vez que fizeres push no GitHub, o site atualiza automaticamente.

### Variáveis de ambiente
Se precisares de `.env`, clica em **"Environment"** e adiciona variáveis.

### Domínio customizado
Clica em **"Settings"** → **"Custom Domain"** para usar o teu próprio domínio.
